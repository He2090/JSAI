# JSAI 工作助手客户使用帮助

本文档面向最终客户和一线使用人员，帮助您快速完成安装、登录、项目创建、模型配置与内容生成。

## 1. 软件简介

JSAI 工作助手当前包含四种主要工作模式：

- `AI漫剧项目`：用于从故事大纲到剧本、角色、分镜、视频的完整工作流制作。
- `文生图`：输入文字提示词，直接生成图片。
- `文生视频`：输入文字提示词，直接生成视频。
- `文图生视频`：上传参考图并输入提示词，生成视频。

## 2. 安装前准备

使用软件前，请确认以下环境已经准备完成：

- 已安装 `JSAI` 客户端。
- 如果使用会员系统，请确认服务端已经启动。
- 如果使用本地模型，请先启动本地服务。

常见本地服务示例：

- 本地文本模型：如 OpenAI 兼容接口、Ollama、LM Studio 等。
- 本地图片模型：ComfyUI。
- 本地视频模型：ComfyUI 对应视频 workflow。

## 3. 客户端首次启动

首次启动客户端时，软件会要求先配置服务器地址。

推荐填写：

- 协议：`http`
- 地址：服务器 IP 或 `127.0.0.1`
- 端口：`5157`

如果服务端部署在其他电脑，请把 `127.0.0.1` 改为服务端机器的实际局域网 IP。

保存后，客户端会自动使用该地址完成：

- 会员登录
- 会员验证
- 版本更新检查

## 4. 登录与会员

### 4.1 登录方式

登录时需要输入：

- 邮箱
- 密码
- 图形验证码

说明：

- 登录成功后，客户端会记住登录状态。
- 下次打开软件时，如果服务器可连接，会自动验证并进入主界面。
- 如果服务器暂时不可用，但本机缓存仍有效、且距离上次联网校验未超过 7 天，软件会自动进入离线模式。

### 4.2 注册方式

注册时需要填写：

- 邮箱
- 密码
- 确认密码
- 邮箱验证码

新用户注册成功后，默认获得试用期。

### 4.3 会员权限

会员信息会显示在主界面右侧，包括：

- 用户名
- 当前套餐
- 到期时间
- 是否允许保存项目

试用期用户可以体验主要功能，但通常不能保存项目。

## 5. 主界面说明

软件首页是黑色工作台界面，进入后可以创建不同类型的项目。

界面主要分为三部分：

- 左侧：项目名称、模组库、历史资产列表。
- 中间：主画布，用于放置节点或直出模组。
- 右侧：节点详情、模型选择、会员信息。

顶部工具栏常见按钮：

- `新建项目`
- `载入项目`
- `保存项目`
- `另存项目`
- `导入 JSON`
- `导出 JSON`
- `模型调用日志`
- `执行工作流`
- `模型设置`

说明：

- `AI漫剧项目` 模式下会显示 `执行工作流`。
- `文生图 / 文生视频 / 文图生视频` 模式下不需要执行整条工作流，因此界面更偏直出创作。

## 6. AI漫剧项目使用流程

AI漫剧项目建议按下面顺序使用。

### 6.1 新建 AI漫剧项目

1. 点击首页的 `AI动漫项目`。
2. 输入项目名称。
3. 在左侧模组库中逐个添加节点到画布。

推荐标准流程：

`故事大纲 -> 故事剧本 -> 创意描述 -> 分镜图片 -> 分镜图拆解 -> 分镜视频 -> 视频合集`

同时：

`故事大纲 -> 角色设计 -> 分镜图片`

### 6.2 故事大纲

在 `故事大纲` 节点中填写：

- 故事简介
- 类型
- 背景
- 风格
- 集数
- 单集时长

点击 `生成大纲` 后，系统会调用文本模型生成结构化大纲。

建议：

- 简介尽量写清楚主角身份、冲突、目标和世界观。
- 如果不满意结果，可使用 `重置大纲` 后重新生成。

### 6.3 故事剧本

在 `故事剧本` 节点中：

- 读取大纲章节
- 选择章节
- 设置拆分集数
- 填写修改建议

点击生成后，系统会为每一集生成创意描述内容。

### 6.4 创意描述

创意描述节点用于显示单集内容，包括：

- 场景描述
- 角色互动
- 动作与冲突
- 对白
- 悬念

点击 `拆分为影视分镜` 后，会在节点内部拆出单条分镜列表。

每条分镜可编辑，方便后续生成分镜图。

### 6.5 角色设计

角色设计节点会从故事大纲中同步角色列表。

每个角色建议按以下顺序操作：

1. `生成角色档案`
2. `生成九宫格`
3. `生成三视图`
4. `查看详情`

说明：

- 九宫格主要用于统一角色脸部、表情和发型。
- 三视图主要用于统一角色服装、轮廓和站姿。
- 查看详情中可看到角色说明、设定文本和图片资产。

### 6.6 分镜图片

分镜图片节点会综合读取：

- 创意描述中的分镜内容
- 角色设计中的角色外观参考

可选择：

- 九宫格（3x3）
- 六宫格（2x3）
- 横屏（16:9）
- 竖屏（9:16）

点击生成后，每一页会生成一张分镜图。

支持功能：

- 上一页 / 下一页
- 放大查看
- 编辑描述

### 6.7 分镜图拆解

分镜图拆解节点会把一整页分镜图拆成单个镜头。

每个镜头通常包含：

- 镜头编号
- 场景
- 画面内容
- 对白
- 景别
- 角度
- 运镜
- 时长

支持：

- 编辑
- 重新获取单个镜头图

### 6.8 分镜视频

分镜视频节点用于把选中的镜头生成成视频。

操作步骤：

1. 在右侧勾选需要输出的分镜镜头。
2. 选择提示词语言。
3. 选择是否需要声音。
4. 点击 `生成提示词`。
5. 选择视频接口、模型、子模型、画幅、时长和画质。
6. 点击 `生成视频`。

说明：

- 文本提示词模型默认优先本地文本模型。
- 视频模型可使用本地模型，也可切换云端模型。
- 云端视频任务提交后，需要等待服务返回结果。
- 成功后，视频会下载到本地，并同步进入 `视频合集`。

### 6.9 视频合集

视频合集节点用于管理已经生成成功的视频片段。

当前可用于：

- 预览单个片段
- 查看片段时长
- 选择参与合集的视频

后续使用建议：

- 按剧情顺序排列片段。
- 确认无误后执行合并输出。

## 7. 文生图使用方法

### 7.1 创建项目

从首页进入 `文生图` 模式后，在左侧添加 `文生图` 节点。

### 7.2 基本操作

在节点中输入：

- 提示词

然后选择：

- 生成模式：普通 / 九宫格 / 三视图
- 比例：如 `16:9`、`1:1`、`9:16`
- 分辨率：如 `普通768`、`普通1024`、`高清2K`、`超清4K`

生成完成后，右侧会显示：

- 图片预览
- 正向提示词
- 反向提示词

支持按钮：

- `打开文件`
- `打开目录`
- `图片保存`
- `清空`

### 7.3 适用场景

文生图适合：

- 角色头像
- 九宫格表情图
- 三视图设定图
- 封面图
- 场景图
- 参考素材

## 8. 文生视频使用方法

文生视频适合直接用一段文字生成视频。

常见步骤：

1. 进入 `文生视频` 模式。
2. 添加 `文生视频` 节点。
3. 输入视频提示词。
4. 选择本地或云端视频模型。
5. 设置画幅、时长、画质等参数。
6. 点击生成。

说明：

- 如果有本地视频模型，默认优先使用本地模型。
- 如果没有本地模型，可切换到云端视频模型。
- 云端模型通常会额外支持大模型和子模型选择。

## 9. 文图生视频使用方法

文图生视频适合：

- 上传一张或多张参考图
- 再用文字描述动作、镜头和氛围
- 生成视频

推荐步骤：

1. 进入 `文图生视频` 模式。
2. 添加 `文图生视频` 节点。
3. 输入文字提示词。
4. 上传参考图片。
5. 选择本地或云端视频模型。
6. 根据当前模型显示的参数界面设置参数。
7. 点击生成。

注意：

- 本地模型和云端模型是两套参数界面。
- 本地模型通常可设置宽高、帧率、时长、保存前缀等。
- 云端模型通常使用画幅、时长、画质、主模型、子模型等参数。

## 10. 模型设置说明

点击顶部 `模型设置` 可进入模型管理界面。

模型分为：

- 本地模型
- 云端模型

模型类别分为：

- 文本模型
- 图片模型
- 视频模型

### 10.1 添加本地文本模型

例如本地 OpenAI 兼容接口：

- 来源：`本地`
- 类别：`Text`
- 地址：本地接口地址
- ID：模型真实名称

### 10.2 添加本地图片模型

例如 ComfyUI：

- 来源：`本地`
- 类别：`Image`
- 地址：如 `http://127.0.0.1:8188/`
- ID：可填写当前 checkpoint 名称，或用于标识的名称

### 10.3 添加本地视频模型

如果视频由 ComfyUI workflow 驱动：

- 来源：`本地`
- 类别：`Video`
- 地址：ComfyUI 地址
- ID：填写对应本地视频模型或 checkpoint 名称

### 10.4 添加云端模型

例如云雾：

- 来源：`云端`
- 类别：`Text / Image / Video`
- 地址：云端 API 地址
- Key：云端 API Key
- ID：云端实际模型 ID

说明：

- 文本模型默认优先本地。
- 图片模型默认优先本地。
- 视频模型如果本地没有可用模型，再切换云端。
- 节点选中后，右侧也可为当前节点单独指定模型。

## 11. 项目保存、载入与导出

### 11.1 保存项目

点击顶部 `保存项目` 可保存当前项目。

项目会保存：

- 节点布局
- 节点参数
- 节点输出
- 部分本地资产路径

### 11.2 另存项目

点击 `另存项目` 后，可手动选择新的保存位置。

### 11.3 载入项目

点击 `载入项目`，选择已有项目文件后，即可恢复工作现场。

项目文件后缀为：

- `.myai`

### 11.4 导入 / 导出 JSON

适合用于：

- 工作流交换
- 备份节点结构
- 技术调试

## 13. 常见问题

### 13.1 客户端无法连接服务器

请检查：

- 服务端是否已经启动
- 服务端地址是否填写正确
- 协议和端口是否正确

本地部署推荐：

- `http://127.0.0.1:5157`

### 13.2 登录后无法保存项目

请检查当前账号是否：

- 仍在试用期
- 会员是否已过期
- 右侧会员信息中的保存权限是否为 `允许`

### 13.3 本地模型调用失败

请检查：

- 本地模型服务是否启动
- 地址是否填写正确
- 模型 ID 是否填写正确
- 端口是否被防火墙拦截

### 13.4 云端模型调用失败

请检查：

- API Key 是否正确
- 账户额度是否充足
- 所选模型是否仍可用
- 网络是否可正常访问云端接口

### 13.5 云端后台显示成功，但软件没有拿到结果

建议检查：

- 任务 ID 是否已经记录
- 是否点击了按 ID 获取
- 网络是否中断
- 本地保存目录是否可写

## 14. 推荐使用顺序

如果您是首次使用，建议按照下面顺序熟悉：

1. 先测试 `文生图`
2. 再测试 `文生视频`
3. 再测试 `文图生视频`
4. 最后进入 `AI漫剧项目` 完整制作

这样可以先确认：

- 登录是否正常
- 模型配置是否正常
- 本地和云端接口是否通畅
